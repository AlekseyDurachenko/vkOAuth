vkoauth

утилита для авторизации приложений ВКонтакте

О проекте

Данная утилита позволяет произвести авторизацию приложений ВКонтакте через протокол OAuth 2.0.

Подробно протокол авторизации описан здесь: https://vk.com/dev/authentication

Инструкция

1. Укажите версию API, ID приложения и разрешения, которые необходимо ему предоставить.

2. нажмите кнопку "Start authorization"(Начать авторизацию).

3. Введите логин и пароль для сайта ВКонтакте

4. Подтвердить разрешения для приложения

После успешной авторизации Вы увидите следующее сообщение

  • user_id - уникальный идентификатор пользователя;
  • access_token - токен доступа для вашего приложения;
  • expires_in - срок годности токена, в секундах. Если равен 0, то срок годности не ограничен. Чтобы получить 0, необходимо выбрать разрешение "offline".
